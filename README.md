# Calorie Tracker (Calorie & Activity Tracker)

A full-stack fitness tracking web application that allows users to log daily calories, steps, and weight, visualize trends, and monitor weekly performance.

## Live Demo
https://calorie-tracker-sand-nine.vercel.app 

## Features

- User authentication using JWT (login and registration)
- Add or update daily fitness records
- One record per day with automatic updates for existing entries
- Interactive charts for steps, calories, and weight trends
- Weekly averages displayed on the dashboard
- Responsive and modern user interface
- Delete records functionality
- Real-time UI updates after data changes

---

## Tech Stack

Frontend:
- React
- Axios
- Recharts

Backend:
- Node.js
- Express
- JWT Authentication
- bcrypt for password hashing

Database:
- PostgreSQL

Deployment:
- Vercel (Frontend)
- Render (Backend)
- Neon (Database)

---

## Key Concepts Implemented

- RESTful API design
- Token-based authentication
- Secure password handling with hashing
- CRUD operations with PostgreSQL
- Data visualization with charts
- React state management using hooks
- Client-server integration
